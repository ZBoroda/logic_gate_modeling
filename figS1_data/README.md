# figS1_data

Simulation results for Fig S1: number of mutations to reach the AND function via random walk (no selection), starting from the simple initial network (each gate takes I1 and I2 as inputs, output connected to gate 2).

500 trials per network size. Generated by `run_figS1.py` using `FastCircuit` (Numba-JIT).

## Files

| File | Contents |
|------|----------|
| `sizes.pickle` | List of 20 network sizes N: `[3, 5, 7, 10, 15, 20, 30, 40, 60, 80, 120, 160, 240, 320, 480, 640, 1000, 1500, 2000, 3000]` |
| `all_mutations.pickle` | List of 20 lists, each with 500 raw mutation counts (one per trial) |
| `means_mutations.pickle` | Mean mutations per N |
| `medians_mutations.pickle` | Median mutations per N |
| `stds_mutations.pickle` | Standard deviation per N |
| `pct5_mutations.pickle` | 5th percentile per N |
| `pct95_mutations.pickle` | 95th percentile per N |

All files are Python pickles. Load with:

```python
import pickle, numpy as np

def load(name):
    with open(f'figS1_data/{name}.pickle', 'rb') as f:
        return pickle.load(f)

sizes         = np.array(load('sizes'))
all_mutations = load('all_mutations')   # list of 500-element lists, one per N
means         = np.array(load('means_mutations'))
medians       = np.array(load('medians_mutations'))
pct5          = np.array(load('pct5_mutations'))
pct95         = np.array(load('pct95_mutations'))
```

## Reproduce or extend

To regenerate the plot:
```bash
conda activate logic_gate
python plot_figS1.py
```

To add more network sizes, edit the `all_sizes` list in `run_figS1.py` and run:
```bash
python run_figS1.py   # skips already-computed sizes automatically
```
